### Game Loop System

The game uses a requestAnimationFrame-based loop for smooth 60fps updates:

```javascript
// In DogeMinerGame constructor
this.startGameLoop();

startGameLoop() {
    const gameLoop = () => {
        if (this.isPlaying) {
            this.update();
            this.render();
        }
        requestAnimationFrame(gameLoop);
    };
    gameLoop();
}
```
